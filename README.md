## Getting Started

프로젝트를 pull 혹은 다운로드 하시고, 해당 프로젝트 디렉토리로 들어갑니다

```bash
flutter pub get
# or
flutter pub add
```
완료되면면
```bash
flutter run
# or
flutter run -d <DEVICE>
```

더 많은 정보는 아래를 참고해주세요

- [Navigation and routing](https://docs.flutter.dev/ui/navigation)
- [Navigator 클래스](https://api.flutter.dev/flutter/widgets/Navigator-class.html)
